Dependency-analysis is a tool suite helping developpers to analyse and maintain quality on their project, including:
* Dependency detection
* Dependency guard
* Cyclic dependencies detection
* Abstractness and instability metrics computation
* Dependency graph drawing.

It's own dependencies are:
 * Turtle: http://sourceforge.net/apps/mediawiki/turtle/index.php?title=Turtle
 * Boost: http://www.boost.org
 * [XeuMeuLeu: http://sourceforge.net/apps/mediawiki/xeumeuleu/index.php?title=Welcome
 * Apache Xerces: http://xerces.apache.org/xerces-c
 * GraphViz: http://www.graphviz.org.

See http://code.google.com/p/dependency-analysis/ for more informations and download.
